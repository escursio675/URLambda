# Feature List
1. Core Functionalities (Abhimanyu): completed on 30th April, 11:40

- The user is able to enter a long URL in a text box
- The user is able to convert the long URL to short URL by clicking the “Generate URL” button
- The user is able to view and copy(using the mouse), the short URL generated
- The user is able to enter the short URL into the browser to visit the same site as that of the long URL

1. QR Code Generation (Moitrayan): completed on 3rd May, 19:32

- The user is able to generate a QR Code for the shortened URL.
- The user is able to view the generated QR Code of the shortened URL.

1. Copy URL button (Abhimanyu): completed on 5th May, 10:03

- The user is able to click on “Copy” button to copy the URL onto their clipboard
- The user is able to verify whether the copy was successful or if there was an error through dynamic button text

1. QR Code Download button (Anuvab): completed on 5th May, 17:50
    - The user is able to download the generated QR Code of the short URL in png format directly into their device.
2. Authentication + Authorization + Dashboard (COMPLETED: 10th May, 04:35)

- The user is able to view details of shortened URLs connected to their profile, delete records, add more records connected to their profile, and view everything in an organized manner through a dashboard.
